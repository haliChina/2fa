工具简介

2FA Authenticator 是一个浏览器端双重验证工具，适合需要快速查看一次性验证码的个人开发者、跨境卖家与运维人员。你可以手动输入 Base32 密钥或扫描二维码，系统会在本地计算 TOTP，不上传密钥到服务器。

适用场景

  • 临时登录云服务、邮箱、社交媒体等开启 2FA 的平台。
  • 需要离线保存多个账户验证码并快速切换查看。
  • 对数据隐私敏感，希望验证码在本地设备内完成计算。

常见问题 FAQ

这个工具会上传我的密钥吗？

不会。验证码计算使用浏览器 Web Crypto API 在本地完成，账户数据默认保存在你的本地存储中，用于下次打开页面继续使用。

支持哪些验证码标准？

当前主要支持 Google Authenticator 兼容的 TOTP（30 秒刷新）格式，输入 Base32 密钥即可生成 6 位动态验证码。
